    Pet & Child Tracker

    Bachelor's FYP

    Raspberry Pi, Python, Telegram bot (Telepot library), MySQL
    • Tracking device for pets and kids
    • Uses Raspberry Pi and its own telegram bot to communicate with it
    • Can be used to get the tracker's location, set a safe zone for the tracker where if it leaves the zone, the person's telegram bot will be spammed with the tracker's last location
    • The person can also ring a buzzer from the telegram bot and access emergency contacts saved through the telegram bot.
    • GitHub Code Here!

    WorkWakeMeUpPls Telegram bot

    Personal Project, you can search it on telegram by finding its username - @WorkWakeMeUpPlsbot

    Python, MongoDB, Telegram bot (Telepot library), Heroku
    • Telegram bot that act as alarm clock to wake people up
    • To stop the alarm, the user needs to answer the given mathematical question, making it hard to snooze
    • Can also include with a creepy image to make sure the person don't go back to sleep (False by default)
    • GitHub Code Here!

    inpt (Python Package)

    Personal Python Package

    Python, PyPI
    • Python package to help simplify the use of inputs with data types
    • Package and Tutorial Here!

    Qadhaa Reminder Telegram bot

    Competition / Personal Project, you can search it on telegram by finding its username - @QadhaaBot

    Python, Telegram bot (Telepot library), MongoDB, Heroku
    • Telegram bot created to keep track of Islamic prayers missed so that it will be easier to be reminded to replace them.
    • Can also choose to send a prayer (Doa/Dua) and Islamic quotes everyday at 8AM and 9PM using the Singapore Timezone
    • GitHub Code Here!
